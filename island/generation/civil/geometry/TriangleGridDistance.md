`dx = abs(ax - bx)`
`dy = abs(ay - by)`

- When cells are on adjacent rows (`dy == 1`), distance is `dx + dy`
  - except when `dx == 0` (or `dx < dy`) and `ax % 2 == ay % 2`, then distance `dy + 2` (or `dx + dy + 2`)

    - If `ax == bx` then distance is `dx + dy + 2`

- When cells are 3 rows apart (`dy == 3`),
  - if `dx >= dy` then distance is `dx + dy`
  - else when `dx < dy` then:
    - if `dy - dx == 1` then:
      - if `ax % 2 == ay % 2` then distance is `dx + dy + 2`
      - else distance is `dx + dy`
    - if `dy - dx == 2` then distance is `dx + dy + 2`
    - if `dy - dx == 3` then:
      - if `ax % 2 == ay % 2` then distance is `dx + dy + 4`
      - else distance is `dx + dy + 2`

Need more data to find a decent algorithm.
What seems to be clear though is if the difference in x is equal or more than the difference in y,
then the simple rule is distance = `dx + dy`.

#!/usr/bin/env python3

def _get_cell_distance_between_cells_by_indices(ax:int, ay:int, bx:int, by:int) -> int:
    dx: int = abs(ax - bx)
    dy: int = abs(ay - by)

    if dx >= dy:
        return dx + dy

    odd_ax: bool = (ax % 2 == 1)
    even_ax: bool = (ax % 2 == 0)
    even_ay: bool = (ay % 2 == 0)
    odd_bx: bool = (bx % 2 == 1)
    even_bx: bool = (bx % 2 == 0)
    even_by: bool = (by % 2 == 0)
 
    if even_ay:
        if even_by:
            if even_ax and odd_bx:
                return dy * 2 + 1
            if odd_ax and even_bx:
                return dy * 2 - 1
        
        else:
            if even_ax and even_bx:
                return dy * 2 + 1
            if odd_ax and odd_bx:
                return dy * 2 - 1

    else:
        if even_by:
            if odd_ax and odd_bx:
                return dy * 2 + 1
            if even_ax and even_bx:
                return dy * 2 - 1
        
        else:
            if odd_ax and even_bx:
                return dy * 2 + 1
            if even_ax and odd_bx:
                return dy * 2 - 1

    return dy * 2

a_rows: list[tuple[int, int]] = [(x, 1) for x in range(13)]
b_cols: list[tuple[int, int]] = [(x, 5) for x in range(13)]

for a in a_rows:
